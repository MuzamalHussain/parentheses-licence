# Parentheses Licence SDK Foundation

This directory contains the official SDK architecture for the Parentheses Licence Public API.

Implemented foundations:

- JavaScript and Node.js client
- TypeScript declarations
- PHP client prepared for WordPress integrations
- Shared API version and SDK version metadata
- Request retries, timeouts, JSON parsing, pagination helpers, and typed JavaScript errors

Prepared future SDK targets:

- Python
- Go
- C#

The SDKs wrap `/api/public/v1` and do not replace or redesign backend API routes.
