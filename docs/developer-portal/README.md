# Parentheses Licence Developer Portal

The Developer Portal is the official documentation and sandbox layer for the Public REST API.

It includes:

- API reference generated from endpoint metadata
- OpenAPI 3.0 specification
- Postman collection and environment metadata
- SDK catalog
- Webhook event reference
- Error and rate limit reference
- Sandbox request explorer

The portal does not expose API secrets. Sandbox execution requires a sandbox API key.
